# PySpark Streamlit Data Analytics Application

## Project Overview

This project is an end-to-end data analytics application developed using Apache Spark (PySpark) and Streamlit.

The system enables users to:

* Upload datasets
* Perform data preprocessing
* Clean and transform data
* Apply feature engineering
* Generate exploratory data analysis (EDA) visualizations

The application combines PySpark's distributed processing capabilities with Streamlit's interactive web interface.

## Technologies Used

* Apache Spark (PySpark)
